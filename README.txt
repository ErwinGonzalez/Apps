Lista de aplicaciones 
Firebase Chat:
	Aplicacion que usa las caracteristicas de tiempo real de firebase para crear permitir la comunicacion entre dos personas

Helpmate
	Aplicacion asistente, permite crear horarios, tareas, listas, recordatorios
KotlinContactList
	Aplicacion de lista de contactos
	Desarrollada en Kotlin uso de LiveData, Room Database 
	Uso de notificaciones
todolist
	Lista de tareas pendientes, permite marcar y desmarcar tareas
TwitterClientApp
	App que hace uso del API de tweeter
	Obtiene la lista de tweets, despliega las imagenes contenidas, o los hashtags utilizados
	Uso del patron MVP para modularizar el app
	Librerias usadas Butterknife, Retrofit, Dagger
